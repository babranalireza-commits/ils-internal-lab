# ILS Internal Lab GitHub Pages Starter

Upload these files to a public GitHub repository:

- `index.html`
- `assets/ILS_token_icon_32x32.svg`

Then enable GitHub Pages from:
Settings → Pages → Build and deployment → Source: Deploy from a branch → Branch: main → Folder: /root → Save

After GitHub Pages is live, use:

Official Project Website:
`https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/`

32x32 SVG icon URL:
`https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/assets/ILS_token_icon_32x32.svg`
